SWA Client Principal Header Repro

x-ms-client-principal header

Login Logout
  • Direct request to API route: includes header
  • Request to route that is explicitly rewritten to the API route: includes header
  • Request to route that is behind allowedRoles but is rewritten via navigationFallback: includes header
  • Request to route that does not have allowedRoles and is rewritten via navigationFallback: does not include header.
